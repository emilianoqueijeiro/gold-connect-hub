ServiceConnect - WordPress Migration Files
=============================================

This package contains clean HTML and CSS files optimized for WordPress migration with Gutenberg Block Editor compatibility.

FILES INCLUDED:
- index.html - Main HTML file with semantic structure
- styles.css - Complete CSS styling system
- README.txt - This instruction file

FEATURES:
✓ Semantic HTML5 structure (<header>, <main>, <section>, <article>, <footer>)
✓ Clean, minimal CSS with HSL color system
✓ Mobile-responsive design
✓ SEO-optimized with proper heading hierarchy
✓ Gutenberg-friendly section structure
✓ WordPress migration ready
✓ Placeholder content for easy customization

WORDPRESS MIGRATION TIPS:
1. Each <section> can be converted to individual Gutenberg blocks
2. Copy CSS variables to your WordPress theme's style.css
3. Replace placeholder images with WordPress media uploads
4. Convert forms to Contact Form 7 or similar plugins
5. Use the CSS classes as custom CSS in WordPress customizer

COLOR SYSTEM:
- Primary: Gold (#FFD700)
- Background: White (#FFFFFF)
- Text: Dark gray (#171717)
- Accents: Light gold backgrounds

SECTIONS INCLUDED:
1. Header with navigation
2. Hero section with search form
3. How It Works (3-step process)
4. Popular Services grid
5. Trust indicators and stats
6. Customer testimonials
7. For Professionals section
8. Footer with links

For questions or support, refer to WordPress documentation for theme customization and Gutenberg block editor usage.
